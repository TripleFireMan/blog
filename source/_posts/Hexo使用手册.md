## Hexo手册

### 1.启动本地调试服务器

```objective-c
hexo server
```

### 2.新建文章

```objective-c
hexo new '文章标题'
```

### 3. 静态文件生成

```objective-c
hexo generate
```

### 4. 文章发布

```objective-c
hexo deploy
```
